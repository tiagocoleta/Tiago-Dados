# Tiago-Dados
Projetos para aprendizado

Projeto 1: O "Legacy Data Cleaner" (Python/Pandas) 
ETL: Tratamento de Dados de Sistemas Legados
Contexto: Simulação de extração de dados de um sistema legado de gestão pública (formato .csv ou .txt) com inconsistências de formatação, datas e valores nulos.

O que este projeto resolve:

Limpeza de ruídos em strings (espaços extras, caracteres especiais).

Padronização de datas para o formato ISO (essencial para bancos de dados modernos).

Tratamento de valores nulos (Imputação de dados).

Exportação para um formato otimizado (Parquet).

Tecnologias: Python, Pandas.


Projeto 2: O "Data Integrator" (SQL/Python)
Objetivo: Mostrar como unir bases de fontes diferentes (ex: RH e Financeiro) para criar uma visão única.

Integração de Bases Heterogêneas (Centralização)
Contexto: Demonstração técnica de como unir dados de cidadãos (Base A) com dados de serviços utilizados (Base B) para gerar um relatório unificado de atendimento.

O que este projeto resolve:

Relacionamento entre tabelas (Joins).

Identificação de chaves primárias e estrangeiras.

Criação de uma "Golden Record" (Registro único da verdade).

Tecnologias: SQL (SQLite), Python.
